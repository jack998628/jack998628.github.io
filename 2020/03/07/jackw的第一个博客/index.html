这次通过搭建hexo博客,学到了一些东西比如:md语法,还有最重要的就是hexo搭建过程中的一些错误.
在这次博客搭建的过程中,我大概本人搭建了4次,因为中间都有一些问题所以说只能删除文件夹,错了没事
重来,而且我也从每一次的错误中吸取了教训,下面我就来总结一下.

总结

  1. 就是在安装npm install的时候可能会很慢,建议去百度npm淘宝镜像,然后再cmd中加进去接下来就会让
    安装快很多,注意:安装之后开头要cnpm。
  2. 就是在安装deploy-git的时候很可能会安装不上导致部署错误,这是不要慌把这段命令npm install hexo-deployer-git –save加上去就可以了,安装完之后你就可以部署了。
  3. 就是主题,你可以根据你的喜好去百度查找喜欢的主题然后下载,在配置文件修改就行了,然后清除缓存,重新部署,你就可以看到新的主题。
  4. 就是写hexo博客的话一定要去看看md语法,不然不知道怎么写博客。
